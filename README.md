# 德庆龙母祖庙三维云游

以单页 HTML 和 Three.js 构建的德庆悦城龙母祖庙三维云游网站。

## 导览功能

- 可旋转、缩放和点击建筑的三维祖庙场景
- 10 处景点悬浮标记与景点资料卡
- 顶部景点全文搜索
- 中轴、院落、水脉、古建分类筛选
- 平面导览与三维沉浸视角切换
- 祖庙精华路线及自动镜头巡游
- 入口定位、缩放、祈愿和票务提示
- 针对手机屏幕优化的底部分类栏与弹出式导览

## 预览

直接打开 `index.html`，或使用 Node.js 启动本地静态服务器：

```bash
node server.js
```

然后访问 `http://localhost:8000/`。

网站通过 jsDelivr 加载 Three.js，预览时需要保持网络连接。
